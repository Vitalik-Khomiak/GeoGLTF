# GeoGLTF Viewer

Дипломний проєкт 2026 ArtLab.

Статичний веб-переглядач `.glb` моделей для навчальних сценаріїв з математики. Додаток побудований на `HTML + CSS + JavaScript + Three.js`, підтримує локальне завантаження 3D-моделі, координатну сітку, осі `X/Y/Z`, обертання, панорамування, масштабування та бібліотеку моделей для швидкого перемикання.

## Швидкий запуск локально

1. Відкрийте PowerShell у каталозі проєкту:

```powershell
cd path\to\GeoGLTF
```

2. Запустіть простий локальний сервер:

```powershell
python -m http.server 8000
```

3. Відкрийте у браузері:

[http://localhost:8000](http://localhost:8000)

> Важливо: додаток використовує ES-модулі, тому відкривати `index.html` напряму через `file://` не варто. Потрібен локальний веб-сервер.

## Як розшарити учням

- Найпростіший варіант: завантажити ці файли у GitHub репозиторій та опублікувати через GitHub Pages.
- Альтернатива: задеплоїти статичний сайт на Netlify або Vercel без серверної логіки.
- Учні зможуть перейти за URL, після чого відкривати локальні `.glb` файли прямо у браузері.

## Як зробити пачку 3D-асетів

1. Поклади свої `.glb` моделі в папку `assets/models/`.
2. Опиши їх у файлі `assets/library.json`.
3. Після деплою ці моделі з’являться у блоці `Опублікована колекція`, і кожен учень зможе обирати, яку модель крутити.

Приклад `assets/library.json`:

```json
{
  "assets": [
    {
      "title": "Куб",
      "description": "Модель для теми об'єму та площі поверхні",
      "file": "./assets/models/cube.glb",
      "sizeLabel": "320 KB"
    },
    {
      "title": "Призма",
      "description": "Модель для вправ із просторового мислення",
      "file": "./assets/models/prism.glb"
    }
  ]
}
```

Окремо є `Локальна колекція`: якщо додати кілька файлів через кнопку або drag-and-drop, вони теж з’являться в списку, але лише на час поточної сесії браузера.

## Майбутня інтеграція з GeoGebra API

Для інтеграції координат вершин із GeoGebra варто додати окремий модуль, який:

1. Обходить геометрію моделі через `mesh.geometry.attributes.position`.
2. Перетворює локальні координати вершин у світові через `mesh.localToWorld(...)`.
3. Формує масив точок у форматі:

```js
[
  { name: "A", x: 1.2, y: 0.5, z: -2.1 },
  { name: "B", x: 2.0, y: 1.1, z: -1.4 }
]
```

4. Передає ці точки в GeoGebra через JavaScript API, наприклад через `ggbApplet.evalCommand(...)`:

```js
ggbApplet.evalCommand("A = (1.2, 0.5, -2.1)");
ggbApplet.evalCommand("B = (2.0, 1.1, -1.4)");
```

Якщо знадобиться, наступним кроком можна додати:

- кнопку `Експортувати вершини`;
- фільтр лише для ключових точок;
- синхронізацію вибраної вершини між Three.js-сценою та GeoGebra 3D.
